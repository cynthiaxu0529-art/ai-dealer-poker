# 🚀 AI Dealer Poker - 快速开始指南

> 5 分钟上手 AI 荷官德州扑克

---

## 第一步：访问

打开浏览器，访问：

```
https://web-production-2a7cc.up.railway.app/
```

---

## 第二步：创建房间

### 如果你是房主：

1. **输入房间名称**（可选）
   ```
   例如：2026-02-14 小龙虾德州夜
   ```

2. **输入你的昵称**
   ```
   例如：小龙虾
   ```

3. **点击 "创建房间"**

4. **复制房间号**（6位字母数字）
   ```
   例如：AB12C3
   ```

5. **分享给好友**
   - 微信、Telegram、Slack 等
   - 好友输入房间号加入

---

## 第三步：加入房间

### 如果你是玩家：

1. **输入房间号**
   ```
   例如：AB12C3
   ```

2. **输入你的昵称**
   ```
   例如：大鲨鱼
   ```

3. **点击 "加入房间"**

---

## 第四步：开始游戏

### 游戏流程：

```
1. 每局结束
   ↓
2. 有人赢得筹码
   ↓
3. 房主记录：谁赢了多少
   ↓
4. 有人买入筹码
   ↓
5. 房主记录：谁买了多少
   ↓
6. AI 荷官解说
   ↓
7. 重复直到结束
```

### 操作按钮：

| 按钮 | 用途 |
|------|------|
| 💰 买入筹码 | 记录买入金额 |
| 📝 记录输赢 | 记录单局输赢 |
| 🏁 结束牌局 | 录入剩余筹码，结算 |

---

## 第五步：结算

### 当所有局结束：

1. **点击 "结束牌局"**

2. **输入每人剩余筹码**
   ```
   小龙虾: 180
   大鲨鱼: 120
   运气王: 150
   ```

3. **系统自动计算**
   ```
   买入 - 剩余 = 输赢
   
   小龙虾: 100 - 180 = +80 🏆
   大鲨鱼: 200 - 120 = -80 💔
   运气王: 150 - 150 = 0 🍀
   ```

4. **AI 荷官总结**
   ```
   本晚共 3 位玩家参与，进行 8 局。
   最大赢家是小龙虾，盈利 +80！
   ```

5. **点击 "复制战绩"**
   - 分享到微信/Telegram

---

## 🎮 完整示例

### 场景：3 人打牌

**开始**:
- 小龙虾创建房间，房间号：`AB12C3`
- 小龙虾分享给大鲨鱼、运气王
- 大鲨鱼、运气王 加入房间

**第一局**:
- 小龙虾赢了 50
- 大鲨鱼输了 30
- 运气王买入 100

**房主操作**:
1. 点击 "📝 记录输赢"
2. 选择 "小龙虾" → "赢得" → 50
3. 点击 "📝 记录输赢"
4. 选择 "大鲨鱼" → "输掉" → 30
5. 点击 "💰 买入筹码"
6. 选择 "运气王" → 100

**AI 荷官解说**:
```
💫 大鲨鱼加入，气氛瞬间紧张！
💎 小龙虾买入 100，牌桌高大上！
🎉 小龙虾赢得 50，运气来了挡不住！
```

**继续...** (重复记录)

**最后结算**:
1. 点击 "🏁 结束牌局"
2. 录入剩余筹码
3. 系统计算输赢
4. 复制战绩

## ❓ 常见问题

### Q: 房间号在哪？
A: 房间页顶部，大字显示，点击可复制

### Q: 能改昵称吗？
A: 目前不能，重新加入会生成新昵称

### Q: AI 荷官会说啥？
A: 买入、输赢时随机解说，结束时有总结

### Q: 数据会丢吗？
A: 关闭浏览器前有效，建议结算完再关

### Q: 能加人多吗？
A: MVP 最多 3 人，更多人开发中
