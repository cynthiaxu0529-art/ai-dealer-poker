## 🚀 使用方式

### Step 1: 访问
```
https://web-production-2a7cc.up.railway.app/
```

### Step 2: 创建房间
1. 输入房间名称（可选）
2. 输入你的昵称
3. 点击"创建房间"
4. 分享房间号给好友

### Step 3: 加入房间
1. 输入 6 位房间号
2. 输入你的昵称
3. 点击"加入房间"

### Step 4: 开始游戏
1. 每局结束，记录买入/输赢
2. AI 荷官实时解说
3. 所有局结束后，录入剩余筹码

### Step 5: 结算
1. 输入每人剩余筹码
2. 系统自动计算输赢
3. AI 荷官总结战局
4. 复制战绩分享

## 📊 数据模型

### 房间 (Room)
```json
{
  "id": "AB12C3",
  "name": "2026-02-14 小龙虾德州夜",
  "status": "playing",
  "players": [...],
  "createdAt": "2026-02-14T05:00:00Z"
}
```

### 玩家 (Player)
```json
{
  "id": "uuid",
  "nickname": "小龙虾 #8888",
  "buyin": 100,
  "finalChips": 180,
  "profit": +80
}
```

### 交易 (Transaction)
```json
{
  "type": "buyin/win/loss",
  "amount": 50,
  "note": "AA 撞 AA，输 100",
  "timestamp": "2026-02-14T05:30:00Z"
}
```
